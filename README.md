# EVE-PRIME

A custom Ubuntu-based Hyprland desktop OS built from scratch.

## What this is

EVE-PRIME is a fully custom Wayland desktop environment built on top of Ubuntu 25.10 and Hyprland 0.54. Every component is purpose-built — nothing is default.

## Features

- **Live wallpaper system** — 108+ video wallpapers via mpvpaper
- **AI-reactive color engine** — extracts palette from wallpaper and propagates to entire desktop in real time
- **Custom login screen** — built in Rust with GTK4, cage compositor, greetd — animated, live wallpaper background
- **Dual waybar** — top status bar + bottom dock, auto-hide, EVE-PRIME themed
- **Full keybind system** — workspace management, scratchpad terminal, clipboard manager, perf overlay
- **Hyprland 0.54** — correct syntax, all plugins loading at parse time

## Stack

Ubuntu 25.10 · Hyprland 0.54 · Rust · GTK4 · Python · Bash · GStreamer · greetd · cage
